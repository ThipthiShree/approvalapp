# ApprovalApp 📱

An Android-based Approval Management App built during my Summer Internship as a Full Stack Developer.

## Features
- View and manage Quotation & Sales Order approval requests
- Approve or Reject requests with a single tap
- Search and filter by Request No, Type, and Status
- View full Approved/Rejected history
- Live location and real-time date in topbar
- Animated ticker showing live pending counts
- Secure login with Change Password support
- Dark mode support

## Tech Stack
- **Frontend:** Kotlin, Android XML
- **Backend:** Node.js, Express.js
- **Database:** Firebase Realtime Database
- **Authentication:** Firebase Auth
- **API:** Retrofit (REST)
- **Deployment:** Railway (24/7 cloud)
